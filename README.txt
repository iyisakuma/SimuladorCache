Para rodar o simulador será necessário possuir um JVM instalado no sistema operacional.
Feito isso, digite a linha de comando 'java -jar cache_simulator.jar <nsets> <bsize> <assoc> <substituicao> <flag_saida> <NOME ARQUIVO> '.
Os arquivos fornecediso para os teste iniciais estão dentro do projeto, caso queira adicionar novos benchmarks coloque na pasta SimuladorCache/src/main/resources/enderecos. 

Outra opção para utilizar o simulador é entrar no diretório 'simulador', adicionar os benchmarks nesse diretório. E utilizar a mesma linha de comando acima.

OBS: o código fonte esta no diretório 'SimuladorCache'.
